# Flappy Rocket

A 3D physics-based rocket navigation game built with Unity. Guide your rocket through obstacles and land safely on the landing pad to progress through levels.

## Gameplay

The player controls a rocket using thrust and rotation. The goal is to navigate through obstacles and reach the landing pad at the end of each level without crashing. The game features 3 levels of increasing difficulty.

- Colliding with obstacles causes the rocket to crash and the level restarts.
- Landing on the finish pad advances to the next level.
- Completing the final level loops back to the first.

## Controls

| Action       | Primary Key | Alternate Key |
| ------------ | ----------- | ------------- |
| Thrust       | Space       | Up Arrow      |
| Rotate Left  | A           | Left Arrow    |
| Rotate Right | D           | Right Arrow   |
| Quit         | Escape      |               |

## Features

- Physics-based rocket movement using Rigidbody forces
- Particle effects for main engine, left thruster, and right thruster
- Explosion and success particle effects on crash and level completion
- Sound effects for engine, crash, and level completion
- Oscillating obstacles that move back and forth
- 3 levels with progressive difficulty
